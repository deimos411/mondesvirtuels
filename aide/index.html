Navigation avec Clavier/Souris :

Instructions

Pour observer votre environnement, maintenez le bouton gauche de la souris enfoncé tout en déplaçant le curseur pour orienter votre vue.
Pour vous déplacer, utilisez les touches directionnelles du clavier ou les touches ZQSD, comme dans un jeu vidéo de type FPS.

Flèches Actions
↑ Avancer
↓ Reculer
← Déplacer vers la gauche
→ Déplacer vers la droite
Touches Actions
Z Avancer
S Reculer
Q Déplacer vers la gauche
D Déplacer vers la droite
Utiliser à la fois les touches de direction et le bouton gauche de la souris pour vous déplacer librement.
Les icones suivantes sont présentes en bas à droite de l'écran :

Icones Actions
Changer de caméra ( touche raccourci C )
Plein écran ( touche raccourci X )
Voler ( touche raccourci F )
Marcher ( touche raccourci M )

Navigation sur Smartphone/Tablette :

Appuyez avec un doigt puis déplacer le sur l'écran pour vous déplacer dans l'environnement 3D.
Appuyez avec deux doigts puis déplacer les pour observer l'environnement et changer l'orientation de la vue.
Des icones sont présentes en bas à droite de l'écran pour vous aider lors de votre exploration :
Icones Actions
Changer de caméra
Plein écran
Voler
Marcher
Vous pouvez utiliser le mode paysage.

Navigation avec Manette Xbox :

Sur ordinateur, utilisez simplement votre manette de jeu Xbox pour naviguer et interagir.
Sur Xbox One / Series S / Series X, ouvrez le navigateur Microsoft Edge sur la console puis accédez à mondesvirtuels.com et choisissez un monde.
Une fois dans un monde, cliquez sur l'icône en forme de manette pour passer du mode de navigation au mode de contrôle de jeu. Pour revenir au mode de navigation, maintenez le bouton Menu de votre manette enfoncé pendant quelques instants.
La plupart des mondes tournent de manière fluide entre 40 et 60 fps même sur la première generation d'Xbox One
